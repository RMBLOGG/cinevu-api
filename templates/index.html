Endpoints

GET /api/movies?page=1
Film Terbaru

Mengambil daftar film terbaru dari halaman utama. Gunakan parameter page untuk navigasi halaman.

Response JSON
{
  "success": true,
  "page": 1,
  "total": 20,
  "next_page": "https://breezyandaman.com/page/2/",
  "results": [
    {
      "url": "https://breezyandaman.com/thriller/the-residence-2025/",
      "title": "The Residence (2025)",
      "poster": "https://...",
      "rating": "6.3",
      "quality": "HD",
      "duration": "110 min",
      "episodes": ""
    }
  ]
}
GET /api/search?q={query}&page=1
Pencarian

Cari film atau series berdasarkan judul. Parameter q wajib diisi.

Response JSON
{
  "success": true,
  "query": "avengers",
  "page": 1,
  "total": 8,
  "results": [ /* array movie card */ ]
}
GET /api/detail?url={url}
Detail Film

Mengambil detail lengkap film termasuk sinopsis, genre, cast, dan daftar episode (untuk series).

Response JSON
{
  "success": true,
  "title": "The Residence (2025)",
  "poster": "https://...",
  "synopsis": "...",
  "rating": "6.3",
  "duration": "110 min",
  "year": "2025",
  "genre": ["Thriller", "Drama"],
  "country": "USA",
  "post_id": "12345",
  "episodes": [
    { "title": "Episode 1", "url": "https://..." }
  ]
}
POST /api/player
Streaming Player

Mengambil daftar server streaming (embed URLs). Memerlukan post_id dan url dari endpoint detail.

📌 Body: { "post_id": "12345", "url": "https://breezyandaman.com/..." }
Response JSON
{
  "success": true,
  "servers": [
    { "server": "p1", "url": "https://embed1.example.com/v/abc" },
    { "server": "p2", "url": "https://embed2.example.com/v/abc" }
  ]
}
GET /api/episode?url={url}
Detail Episode

Detail episode series beserta semua server player yang tersedia. URL diambil dari array episodes di endpoint detail.

Response JSON
{
  "success": true,
  "title": "Episode 1 - S01E01",
  "post_id": "99123",
  "servers": [
    { "server": "p1", "url": "https://embed1.example.com/v/abc" },
    { "server": "p2", "url": "https://embed2.example.com/v/abc" }
  ]
}

Filter

GET /api/genre/{slug}?page=1
Filter Genre

Filter film berdasarkan genre. Slug sesuai kategori di situs.

📌 Slug: action · drama · thriller · horror · comedy · romance · mystery · crime · fantasy · science-fiction · adventure · family · music
Response JSON
{
  "success": true,
  "genre": "action",
  "page": 1,
  "total": 20,
  "next_page": "https://...",
  "results": [ /* array movie card */ ]
}
GET /api/year/{year}?page=1
Filter Tahun

Filter film berdasarkan tahun rilis (2017–2026).

📌 Contoh: /api/year/2025?page=1
Response JSON
{
  "success": true,
  "year": "2025",
  "page": 1,
  "total": 20,
  "next_page": null,
  "results": [ /* array movie card */ ]
}
GET /api/country/{country}?page=1
Filter Negara

Filter film berdasarkan negara. Mapping slug lokal ke slug situs sudah di-handle otomatis.

📌 Slug: indonesia · korea · jepang → japan · usa · china · thailand · filipina → philippines · india
Response JSON
{
  "success": true,
  "country": "jepang",
  "slug": "japan",
  "page": 1,
  "total": 20,
  "results": [ /* array movie card */ ]
}
